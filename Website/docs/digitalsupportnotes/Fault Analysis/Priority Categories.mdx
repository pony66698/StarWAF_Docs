There are different priorities for different issues. 

### Low Priority

- Mouse not working
- Monitor damage
- USB slot not working
- Keyboard key missing
- Slow Computer

### Medium Priority

- One computer not
working
- Sound not working
- Network cable broken
for a device
- Software not working

### High Priority

- Internet outage
- Power outage
- Overload
- Risk of data
corruption
- Server down
