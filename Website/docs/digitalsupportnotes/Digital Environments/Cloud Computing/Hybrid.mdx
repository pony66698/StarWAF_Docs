在这里，您可以了解到您的网站是否合适接入本项目。

### 针对大陆内容分发项目
根据《中华人民共和国宪法》第三十五条、第三十六条、第四十条；《中华人民共和国刑法》第二百三十二条、第二百三十四条、第二百三十五条、第二百四十条、第二百五十条、第二百五十一条；《中华人民共和国网络安全法》第四十四条、第四十六条；《中华人民共和国广告法》第四十二条、第五十三条等相关法律规定，以下类型的网站不支持接入本项目：  
:::info
色情网站：包括涉及淫秽、色情、色情漫画、色情小说、色情影视等违反道德规范和法律法规的内容的网站。

赌博网站：包括提供赌博、彩票、博彩、网络赌博等违反国家法律法规的活动的网站。

毒品网站：包括贩卖、宣传毒品、药物、非法药物的网站，以及吸毒、制毒等违法行为的信息。

政治敏感网站：包括宣传煽动颠覆国家政权、分裂国家、恐怖主义、极端主义等危害国家安全和社会稳定的内容的网站。

盗版资源网站：包括提供盗版软件、盗版音乐、盗版电影、盗版图书等侵犯他人知识产权的网站。

虚假广告网站：包括散布虚假广告、虚假宣传、误导性信息等违反广告法规定的网站。

非法交易网站：包括贩卖非法商品、非法交易、非法传媒、非法文物等违法经营行为的网站。
:::

### 针对亚太内容分发项目
根据《刑法》第175条（公然わいせつ行為等）和《児童買春、児童ポルノ禁止法》、《刑法》第185条（公営ギャンブル等の幇助）、《著作権法》；《情報通信ネットワーク利用促進及び情報保護等に関する法律》和《青少年保護育成条例》；《형법》第246条（협박）和《도박법》、《정보통신망법》；《兒童及少年性交易防制法》、《刑法》第266条（輸贏錢物之博鬥）、《國家安全法》；《印度刑法典》第292条（出售、出租或分发淫秽物品）、《公共博彩法》、《印度刑法典》第153A条（促使与宣传敌对行动）；《香港法例》第147章《儿童保护条例》、《香港法例》第148章《远程博彩条例》、《香港法例》第8章《防止恐怖活动条例》等法律条款，以下类型的网站不支持接入本项目：  
:::info
色情网站：涉及色情或淫秽内容的网站，特别是未经年龄验证或未做出适当限制的情况下向未成年人提供此类内容（特别是向未成年人提供此类内容或包括未经年龄验证或未做出适当限制的情况下向未成年人提供此类内容）

赌博网站：非法赌博相关的网站，包括未经许可的赌博活动和网上赌博平台

盗版资源网站：侵犯他人版权或知识产权的网站，包括盗版和非法传播受版权保护的内容。

政治敏感网站：宣扬恐怖主义、分裂国家或违法行为的网站（或宣扬恐怖主义、分裂国家或违法行为的网站；涉及宗教或社会冲突、恶意宣传或煽动仇恨的网站）

制作、传播恶意软件或病毒：涉及恶意软件、病毒传播或黑客工具的网站。

:::
