如何加入边缘网络加速计划 / 什么是边缘网络加速计划

**边缘网络加速计划（Edge Network Acceleration Program）** 是 **StarWAF Project** 的子项目，旨在  **提高互联网内容和应用程序的交付速度和性能** 边缘网络加速计划利用分布在全球各地的 **边缘节点（Edge Nodes）** 来缩短数据传输的距离，从而减少延迟和网络拥塞。  
**边缘网络加速计划** 需要你的加入以建立分布式的边缘节点，将内容和应用程序的副本存储在离用户更近的位置。


## 如何加入该项目？

### 您需要提供的：

:::info 您需要提供的

1.您需要是 **合法设立** 的 **数据中心类业务/接入类业务** 服务提供商（您需要持有相关证件）  
2.您需要提供给我们您的 **闲置服务器**  用于实现内容分发 / 边缘网络加速  
3.为保证用户信息安全，我们目前 **暂不接受以个人名义加入此计划**

:::

### 在加入此计划后，您可以获得：

:::info 在加入此计划后，您可以获得

1.您可以获得 **StarWAF Project** 的纪念品以表明您是该计划的贡献者  
2.您可以在 **StarWAF Project** 中获得与您为我们提供的服务器售价 **等价值** 的余额用以抵用消费  
3.您可以成为该项目的 **Contribute**

:::
